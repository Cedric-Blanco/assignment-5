How to Apply CSS to HTML

The link tag (pretend that there is a left arrow before link) link rel="stylesheet" href="whateverTheNameIs.CSS" /> can link a CSS file and apply it to an HTML page.
We also need to use href to use the specific CSS file we are trying to use.

How to use JavaScript on a Webpage

We can write JavaScript code within an html document using the tag of script. We can add src after script like (there are left arrows also missing here)
script src = "path/of/the/script/whateverTheNameIs.js"> /script> to link an external JavaScript file to an html page.
